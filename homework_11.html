  1. Look up the geom_segment() function (outside of your Rmd file). Add a geom_segment() that draws a line from each origin to each destination, with transparency mapped to the frequency of that origin-destination pair. Display the resulting plot.

Now, let’s look at our U.S. map with lines from each origin to each destination, with transparency mapped to the frequency of that origin-destination pair:5

ggplot() +
  geom_polygon(
    data = map_data("state"),
    aes(x = long, y = lat, group = group),
    fill = NA, color = "gray"
  ) +
  geom_segment(
    data = freq_tab_dest,
    aes(
      x = origin_lon,
      y = origin_lat,
      xend = dest_lon,
      yend = dest_lat,
      alpha = n
    ),
    color = "blue"
  ) +
  coord_fixed(1.3)
